Install it in four steps

No technical skill needed. Takes about a minute. Do it once.

1

Download & save the file

Click a Download button above and save the .tdl file somewhere you'll remember, for example your Desktop or a folder called Tally TDL.

2

Open the TDL manager in Tally

In TallyPrime: press F1 (Help) → TDL & Add-On → F4 (Manage Local TDLs).
In Tally.ERP 9: F12 → Product & Features → F4.

3

Point it at the file

Set “Load selected TDL files on startup” to Yes, then type or paste the full path to the .tdl file you saved, and press Enter to accept.

4

Export from the main menu

Back on the Gateway of Tally you'll see a new item,DD Data Export. Open it, choose what to export, and your file is written to Tally's folder. Open it in Excel. Done.

Tip: set your company period first with Alt+F2,the export covers exactly the dates you choose. Files are saved in the folder Tally opens from.

Questions

Is it safe? Where does my data go?

Completely offline. The add-on only reads your company data and writes a file to your own computer. Nothing is uploaded anywhere, and it never edits or deletes your Tally data. You can read the entire source code on GitHub.

Which versions of Tally does it work with?

TallyPrime and Tally.ERP 9. A few field names vary slightly between Tally releases; if a column ever looks empty, load the included FieldProbe file once to check which names your build uses (details in the GitHub README).

Where are the exported files saved?

In the folder Tally starts from (its application/data folder). After exporting, open that folder and you'll find files like LedgerVoucher.csv,double-click to open in Excel.

I have lakhs of vouchers, will it handle that?

Yes for a normal year of data. For very large or automated pulls (hundreds of thousands to millions of rows) the GitHub README explains the fastest approach. Tip: export a shorter date range at a time.

Does it cost anything?

No. It's free and open-source under the MIT licence. Use it, share it, or adapt it for your own firm.
